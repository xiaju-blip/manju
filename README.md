# 🎬 manju - 网页AI漫剧生成器

基于免费网页版AI模型生成漫剧、图片、视频的一站式桌面应用，**不需要API密钥**，直接使用浏览器访问各个AI网站生成内容。

## ✨ 特性

- 🆓 **完全免费**：使用各个平台免费网页版，不需要API密钥
- 🖥️ **跨平台桌面应用**：基于Electron，支持Windows/Mac/Linux
- 📖 **漫剧剧本生成**：自动生成完整漫剧剧本，包含故事梗概、分镜、人设
- 🖼️ **文生图**：直接在免费网页AI生成漫画图片
- 🎬 **图生视频**：将图片转换成动态视频
- 🤖 **支持多个免费AI平台：

### 文本AI（生成剧本）
- DeepSeek 网页版
- 通义千问 网页版
- 豆包 网页版
- 智谱清言 网页版
- Kimi 网页版

### 文生图（生成图片）
- 土豆AI绘图
- 文心一格 (百度)
- 通义万相 (阿里)
- Pixlr AI

### 图生视频（生成视频）
- Pika Labs
- Runway ML
- 可灵AI (快手)

- 🖼️ 可自动为每个分镜生成AI绘图提示词，直接用于文生图
- 📋 一键复制结果

## 🚀 快速开始

### 1. 安装依赖

```bash
cd manju
npm install
npx playwright install chromium
```

### 2. 运行应用

```bash
npm start
```

### 3. 打包成可执行文件（可选）

```bash
# 打包成当前系统的安装包
npm run build
```

## 📖 使用流程

1. **漫剧生成：**
   - 点击「启动浏览器」按钮，会自动打开一个Chrome浏览器窗口
   - 在弹出的浏览器中，**手动登录**你使用的AI平台（每个平台只需登录一次，登录后保持打开即可保持登录状态）
   - 回到应用，选择文本AI平台 → 输入故事主题 → 选择漫画风格 → 点击「生成漫剧」
   - 等待1-3分钟，AI生成完成后结果显示在右侧
   - 点击「生成绘图提示词」可为每个分镜生成文生图提示词

2. **文生图：**
   - 切换到「文生图」标签
   - 选择文生图平台 → 输入提示词 → 选择风格 → 点击生成
   - 生成完成后图片会直接显示在结果区域

3. **图生视频：**
   - 切换到「图生视频」标签
   - 选择视频平台 → 输入提示词 → 填入本地图片路径 → 点击生成
   - 需要先在浏览器登录账号

## 🔐 注意事项

- 本应用只是自动化控制浏览器帮你操作，**所有AI内容都来自各个平台的网页版**
- 首次使用需要你手动在浏览器中登录AI账号，本应用**不会存储你的账号密码**
- 请遵守各个AI平台的使用条款
- 生成时间取决于AI响应速度：
  - 漫剧剧本：1-3分钟
  - 文生图：1-5分钟
  - 图生视频：3-10分钟

## 📝 输出格式

生成的漫剧包含：
1. **故事梗概** - 简要描述整个故事
2. **分镜列表** - 每个分镜包含序号、场景描述、对话内容、画面提示
3. **角色设定** - 主要角色人设
4. *(可选)* **AI绘图提示词** - 为每个分镜生成详细的绘图提示词，直接复制到文生图工具

```javascript

    submitSelector: '提交按钮CSS选择器',
    outputSelector: '输出内容CSS选择器',
    waitSelector: '生成中指示器CSS选择器',
    category: 'text'
  }
};

// 文生图平台

    waitSelector: '生成中选择器'
  }
};
```
